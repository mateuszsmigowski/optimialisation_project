# optimialisation_project

Plan projektu
"Optymalizacja składowania produktów w magazynie na półkach typu "high-rack".

Problem:
Jak optymalnie rozmieścić produkty na półkach zlokalizowanych w magazynie, aby zminimalizować czas dostępu do produktów oraz zmaksymalizować wykorzystanie przestrzeni"

1) Optymalizacja czasu dostępu - produkty częściej zamawiane powinny być łatwiej dostępne (składowane na niższych poziomach)
2) Wykorzystanie przestrzeni magazynowej - maksymalizacja wykorzystania dostępnej przestrzeni do magazynowania
3) Minimalizacja kosztów operacyjnych

Rozważane algorytmy do implementacji:

1) Algorytm zachłanny
2) Algorytmy genetyczne
3) ew. Algorytm mrówkowy
